Git Keybind Cheatsheet

Version: 2.40.0

Basic Commands

Initialize repository
Create a new Git repository
git init
Clone repository
Clone a remote repository
git clone
Check status
Show working tree status
git status
Add files
Add files to staging area
git add
Commit changes
Commit staged changes
git commit -m "message"

Branching

Create branch
Create a new branch
git branch
Switch branch
Switch to a branch
git checkout
Create and switch
Create and switch to new branch
git checkout -b
Merge branch
Merge branch into current
git merge
Delete branch
Delete a branch
git branch -d

Remote Operations

Fetch changes
Download changes from remote
git fetch
Pull changes
Fetch and merge changes
git pull
Push changes
Upload changes to remote
git push
Add remote
Add a remote repository
git remote add

History & Inspection

View log
Show commit history
git log
Show differences
Show file differences
git diff
Show commit
Show commit details
git show
Blame file
Show line-by-line history
git blame
