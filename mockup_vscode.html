VSCode Keybind Cheatsheet

Version: 1.85.0

File Operations

Open file
Open a file from the filesystem
Ctrl+O
Save file
Save the current file
Ctrl+S
Save as
Save file with a new name
Ctrl+Shift+S
New file
Ctrl+N
Close file
Ctrl+W

Editing

Copy
Copy selected text
Ctrl+C
Cut
Cut selected text
Ctrl+X
Paste
Paste from clipboard
Ctrl+V
Undo
Ctrl+Z
Redo
Ctrl+Y
Select all
Ctrl+A

Navigation

Go to line
Jump to a specific line number
Ctrl+G
Find
Find text in current file
Ctrl+F
Find and replace
Find and replace text
Ctrl+H
Go to definition
F12
Go back
Alt+Left

Multi-key Commands

Open command palette
Access all available commands
Ctrl+Shift+P
Split editor
Split the editor window
Ctrl+KCtrl+S
Toggle terminal
Show or hide the integrated terminal
Ctrl+Shift+`
